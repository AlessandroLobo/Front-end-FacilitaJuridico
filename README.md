## How to Run the Project

1. Clone the repository:
   ```bash
   git clone https://github.com/AlessandroLobo/Front-end-FacilitaJuridico.git
   ```

2. Clone the repository:

```bash
npm install

npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
3. access
http://localhost:3000
